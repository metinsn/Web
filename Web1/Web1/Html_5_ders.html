DIV,division lelimesinin kısaltmasıdır.HTML sayfaların içinde özel alanlar oluşturmak için kullanılan elementtir. Eskiden tablolar kullanılarak yapılan tasarımlar uzun zaman ve performans kaybına neden olyordu.Div kullanımı bunun önüne geçmektedir.

Bazı özellikleri;

  • id => site içindeki özel yeri temsil etmektedir.ID benzersizdir.W3C standartlarında bir id başka bir elementte kullanılamaz.
  • width => elementin genişliğini temsil eder.
  • height => elementin yüksekliğini temsil eder.
  • margin-left => sol taraftan dış elemente olan uzaklığı temsil eder. margin-right => sağ taraftan dış elemente olan uzaklığı temsil eder.
    Not: bir elementin margin-left ve margin-right özelliklerine "auto" girerseniz, çözünürlüğe göre saqyfanın tam ortasında yer alır.
  • float => tasarımsal alanın hangi yöne olacağını belirtir.
